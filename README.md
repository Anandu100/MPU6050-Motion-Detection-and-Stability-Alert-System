# MPU6050 Motion Detection and Stability Alert System

## 📖 Overview

This project is an Arduino-based motion detection and stability monitoring system that uses the MPU6050 accelerometer and gyroscope sensor. The system continuously monitors the orientation of the device and detects abnormal tilt or movement. When the predefined threshold is exceeded, it activates an LED and a buzzer to provide real-time visual and audio alerts.

## 🚀 Features

- Real-time motion detection
- Tilt and stability monitoring
- LED alert system
- Buzzer alert system
- Arduino Uno implementation
- MPU6050 accelerometer & gyroscope integration

## 🛠 Components Used

- Arduino Uno
- MPU6050 Sensor
- LED
- Buzzer
- Breadboard
- Jumper Wires

## 💻 Technologies

- Arduino IDE
- C/C++
- I2C Communication
- Embedded Systems

## ⚙️ Working

The MPU6050 continuously measures acceleration and angular velocity. The Arduino processes this data to calculate the device's orientation. If the detected tilt exceeds the predefined threshold, the LED lights up and the buzzer is activated. Once the device returns to a stable position, both alerts are automatically turned off.
